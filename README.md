# Unit Testing Example - PHPUnit

## Get Started

### Prerequisites
PHP 7.0+

Composer

### Clone repository
```
git clone
```

### Install with Composer
```
composer install
```
### Run Test

Windows
```
vendor\bin\phpunit --bootstrap vendor\autoload.php tests\CalculatorTest.php
```
Linux & Mac
```
./vendor/bin/phpunit --bootstrap vendor/autoload.php tests/CalculatorTest
```
